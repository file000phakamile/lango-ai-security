Lango — AI Data Guard

Log in with your Lango account to enable prompt scanning on supported sites (ChatGPT, Claude, Gemini, DeepSeek, Copilot — see extension/USER_GUIDE.md for which of these are actually verified working versus implemented-but-unverified).

v0.1. Your JWT is stored in chrome.storage.local for this browser profile only — not synced to other devices. Default API base URL is the live Render backend; override it to http://localhost:8080 to test against a locally-run backend instead (see backend/README / the main repo README's Setup section for running the backend locally). See extension/README.md for full install and testing instructions.
